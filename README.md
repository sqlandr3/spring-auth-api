# Spring Auth API

API REST de autenticação e autorização desenvolvida em Java com Spring Boot, com foco em estudo prático de segurança, controle de acesso e boas práticas de back-end.

Este projeto foi criado como parte do meu aprendizado em desenvolvimento back-end com Java e Spring, consolidando conceitos fundamentais usados em APIs reais.

---

## Objetivo do Projeto

- Compreender como funciona o fluxo de autenticação em uma API REST
- Implementar segurança utilizando Spring Security
- Trabalhar com autenticação baseada em token (JWT)
- Aprender a estruturar um projeto Spring Boot de forma organizada
- Publicar e versionar um projeto profissionalmente no GitHub

---

## Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- Maven
- Git e GitHub

---

## Funcionalidades

- Autenticação de usuários via API REST
- Geração e validação de token JWT
- Proteção de endpoints com Spring Security
- Configuração de filtros de segurança
- Controle de acesso baseado em autenticação

---

## O que eu aprendi com este projeto

Durante o desenvolvimento deste projeto, aprendi na prática:

- Como funciona o ciclo de autenticação em uma API REST
- Diferença entre autenticação e autorização
- Como o Spring Security intercepta requisições HTTP
- Como configurar filtros de segurança e cadeias de autorização
- Como gerar, assinar e validar tokens JWT
- Como proteger endpoints e liberar rotas públicas
- Como estruturar camadas básicas de um projeto Spring Boot
- Como configurar dependências com Maven
- Como versionar um projeto corretamente com Git
- Como publicar e manter um repositório no GitHub de forma profissional

Este projeto consolidou minha base para evoluir para APIs mais complexas, com persistência em banco de dados, roles de usuário e autenticação avançada.
